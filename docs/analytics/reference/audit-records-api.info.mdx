This API allows administrators to retrieve audit records in JSON format.

It retrieves audit records with **create**, **delete** and **update** events on following entities in Platform.
* Call forwarding rules
* Call queue
* Phone Number
* Extension
* Ring group
* User basic information
* Auto attendant basic information

_**For step-by-step instructions on using the Audit API refer to the [Audit API Summary Guide ](https://docs.google.com/document/d/1rtGbpvCHKtR19fqPYbPh-CA6YTQkeubKgDl6r-b9N4M/edit?usp=sharing)**_.
