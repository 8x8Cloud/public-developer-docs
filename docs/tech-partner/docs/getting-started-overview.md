# Overview

Welcome to the 8x8 Technology Partner documentation. This section covers everything you need to integrate your application with 8x8 products, or embed 8x8 products into your own platform.

## Partner Integration Guide

Embed your web application as an iframe panel inside an 8x8 application. Your app runs alongside the agent's active interactions and communicates bidirectionally with the Platform UI Shell and the parent application through the `@8x8/pui-partner-comm` SDK. See the [Partner Integration Guide](../../tech-partner/docs/partner-sdk-integration-guide.mdx).

## Maestro SDK

Embed an 8x8 application inside your own platform and subscribe to UC and CC interaction lifecycle events (offered, accepted, completed, terminated). Use this for use cases such as click-to-call, screen pop, interaction logging, and automated record creation. See the [Maestro SDK documentation](../../tech-partner/docs/partner-sdk-maestro-sdk-overview.mdx).

## UI Guidelines

If your integration includes a user interface, refer to the [UI Guidelines](../../tech-partner/docs/ui-guidelines.mdx) for styling, accessibility, and theme requirements when building within the 8x8 ecosystem.
